Version Control for Beginners

⏱ 30 min🟢 Beginner📁 Git, GitHub

Git কী?

Git হলো একটি version control system। মানে তোমার project এর প্রতিটি পরিবর্তন track করে রাখে। Mistake হলে আগের version এ ফিরে যাওয়া যায়।

GitHub কী?

GitHub হলো একটি online platform যেখানে Git repository host করা যায়। Team এর সাথে code share ও collaborate করা যায়।

Step 1: Git Install করো

git-scm.com থেকে Git download করে install করো। তারপর terminal এ check করো:

Terminal
git --version

Step 2: Git Config করো (একবারই করতে হবে)

Terminal
git config --global user.name "Md. Prince Mahmud"
git config --global user.email "devport910@gmail.com"

Step 3: প্রথম Repository তৈরি করো

Terminal
# Project folder এ যাও
cd my-project

# Git initialize করো
git init

# সব files add করো
git add .

# First commit করো
git commit -m "Initial commit"

Step 4: GitHub এ Push করো

github.com এ নতুন repository তৈরি করো, তারপর:

Terminal
# GitHub repository connect করো
git remote add origin https://github.com/username/repo-name.git

# Code push করো
git branch -M main
git push -u origin main

Step 5: Daily Workflow

Terminal — প্রতিদিনের কাজ
# Changes দেখো
git status

# Files add করো
git add .
# অথবা specific file
git add index.html

# Commit করো
git commit -m "Add login page"

# GitHub এ push করো
git push

গুরুত্বপূর্ণ Git Commands

Commandকাজ
git initনতুন repository তৈরি
git statusকোন files changed দেখাও
git add .সব changes stage করো
git commit -m "msg"Changes save করো
git pushGitHub এ upload করো
git pullGitHub থেকে download করো
git clone URLRepository copy করো
git logCommit history দেখো
git branchসব branches দেখো
git checkout -b nameনতুন branch তৈরি করো

💡 Tip: Commit message সবসময় meaningful লেখো। "Fix login bug" ভালো, "update" খারাপ।
