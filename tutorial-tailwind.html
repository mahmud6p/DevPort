Modern UI Components with Tailwind

⏱ 40 min🟢 Beginner📁 HTML, Tailwind CSS

Tailwind CSS কী?

Tailwind CSS হলো একটি utility-first CSS framework। এতে আলাদা CSS লেখার দরকার নেই — সরাসরি HTML এ class দিয়ে design করা যায়।

Step 1: Tailwind Setup (CDN)

সবচেয়ে সহজ উপায় — HTML এ CDN link যোগ করো:

index.html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Tailwind UI</title>
  <script src="https://cdn.tailwindcss.com"></script>
</head>
<body class="bg-gray-100 p-8">
  <!-- Components এখানে যাবে -->
</body>
</html>

Step 2: Button Components

Buttons
<!-- Primary Button -->
<button class="bg-blue-500 hover:bg-blue-600 text-white 
               font-semibold py-2 px-6 rounded-lg 
               transition duration-200">
  Primary Button
</button>

<!-- Outline Button -->
<button class="border-2 border-blue-500 text-blue-500 
               hover:bg-blue-500 hover:text-white
               font-semibold py-2 px-6 rounded-lg 
               transition duration-200">
  Outline Button
</button>

<!-- Danger Button -->
<button class="bg-red-500 hover:bg-red-600 text-white 
               font-semibold py-2 px-6 rounded-lg 
               transition duration-200">
  Delete
</button>

Grid

💡 Tip: Tailwind এর সব class মনে রাখা সম্ভব না — tailwindcss.com/docs সবসময় খোলা রাখো!

গুরুত্বপূর্ণ Tailwind Classes

  • Spacing: p-4, px-6, py-2, m-4, mt-8, gap-4
  • Colors: bg-blue-500, text-gray-800, border-red-400
  • Flex: flex, items-center, justify-between, gap-4
  • Grid: grid, grid-cols-3, col-span-2
  • Responsive: md:flex, lg:grid-cols-3, sm:hidden
  • Hover: hover:bg-blue-600, hover:text-white
